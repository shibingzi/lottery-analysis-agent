# 彩票分析助手 - 网站版

## 🎱 项目简介

这是一个基于Web的彩票数据分析平台，提供直观的可视化界面来分析双色球和大乐透的历史数据。

## 🚀 快速开始

### 方法一：使用Python HTTP服务器（推荐）

```bash
# 进入网站目录
cd web

# 启动HTTP服务器（Python 3）
python -m http.server 8080

# 或者使用Python 2
python -m SimpleHTTPServer 8080
```

然后打开浏览器访问：`http://localhost:8080`

### 方法二：使用Node.js（如果你已安装）

```bash
# 安装http-server（全局安装，只需一次）
npm install -g http-server

# 进入网站目录并启动
cd web
http-server -p 8080
```

### 方法三：使用VS Code Live Server

1. 安装VS Code的Live Server插件
2. 右键点击 `index.html` 文件
3. 选择 "Open with Live Server"

## ⚠️ 免责声明

**本网站所有分析仅供娱乐参考，不构成投注建议。**

- 彩票开奖是完全随机的独立事件
- 历史数据对未来开奖没有任何预测价值
- 每个号码在每期中奖的概率是相等的

## 📝 使用说明

### 查看数据分析
1. 点击导航栏"数据分析"
2. 选择彩种（双色球/大乐透）
3. 选择分析期数
4. 点击"运行分析"按钮
5. 查看各种统计图表和数据

### 固定号码分析
1. 点击导航栏"固定号码"
2. 选择彩种类型
3. 输入您的固定号码
4. 点击"分析号码"查看历史表现
5. 点击"生成组合"获取推荐组合

### 查看历史报告
1. 点击导航栏"历史报告"
2. 浏览报告列表
3. 点击"查看报告"查看详情

## ⌨️ 常用命令

### 基础查询

| 命令 | 说明 |
|------|------|
| `查看[彩种]最新开奖` | 查看最新一期开奖结果 |
| `查看[彩种]最近N期` | 查看最近N期开奖记录 |
| `查看[彩种]第XXX期` | 查看指定期号开奖结果 |

**示例**:
```
查看双色球最新开奖
查看大乐透最近20期
查看双色球第2025023期
```

### 统计分析

| 命令 | 说明 |
|------|------|
| `分析[彩种]热号冷号` | 统计热号冷号 |
| `分析[彩种]遗漏值` | 统计号码遗漏情况 |
| `分析[彩种]奇偶比` | 统计奇偶分布 |
| `分析[彩种]大小比` | 统计大小分布 |
| `分析[彩种]连号` | 统计连号出现情况 |
| `生成[彩种]完整报告` | 生成综合统计报告 |

**示例**:
```
分析双色球热号冷号
分析大乐透最近100期的遗漏值
生成双色球完整统计报告
```

### 固定号码分析

| 命令格式 | 说明 |
|----------|------|
| `分析固定号码 [红球] + [蓝球]` | 双色球固定号码分析 |
| `分析固定号码 [前区] + [后区]` | 大乐透固定号码分析 |

**示例**:
```
分析固定号码 07 18 25 + 14
分析固定号码 05 12 23 28 + 03 09
```

### 号码生成

| 命令 | 说明 |
|------|------|
| `机选[N]注[彩种]` | 随机生成N注号码 |
| `生成[彩种]号码` | 生成一注号码 |

**示例**:
```
机选5注双色球
生成大乐透号码
```

## 📊 功能详解

### 1️⃣ 查看最新开奖

**功能**: 显示最新一期的开奖结果

**输出内容**:
- 期号
- 开奖日期
- 开奖号码（红球/蓝球或前区/后区）
- 中奖情况（一等奖注数、奖池金额等）
- 数据来源和查询时间

**示例对话**:
```
用户: 查看双色球最新开奖

Agent: 
## 双色球最新开奖结果

期号: 2025023
开奖日期: 2025-03-02

红球: 03 07 12 18 25 30
蓝球: 14

一等奖: 5注，每注1000万元
二等奖: 89注，每注28.5万元
奖池: 23.45亿元

数据来源: 中国福彩官网
查询时间: 2026-02-07 15:30:00
```

---

### 2️⃣ 热号冷号分析

**功能**: 统计指定周期内各号码的出现频率

**输出内容**:
- 热号 TOP5（出现次数最多的号码）
- 冷号 TOP5（出现次数最少的号码）
- 出现频率图表
- 统计周期说明

**示例对话**:
```
用户: 分析双色球最近100期的热号冷号

2. 23号 - 出现16次
3. 07号 - 出现15次
4. 18号 - 出现14次
5. 30号 - 出现13次

### 冷号 TOP5
1. 01号 - 出现3次
2. 33号 - 出现4次
3. 11号 - 出现5次
4. 28号 - 出现6次
5. 04号 - 出现7次

⚠️ 注: 热号/冷号仅为历史统计，不代表未来中奖概率
```

### 3️⃣ 遗漏值分析

**功能**: 统计各号码自上次开出以来的遗漏期数

**输出内容**:
- 最大遗漏 TOP5
- 当前遗漏分布
- 遗漏值趋势图

**示例对话**:
```
用户: 分析双色球遗漏值

Agent:
## 双色球遗漏值分析

### 最大遗漏 TOP5
1. 01号 - 遗漏45期
2. 33号 - 遗漏38期
3. 11号 - 遗漏32期
4. 28号 - 遗漏28期
5. 04号 - 遗漏25期

### 遗漏分布
- 热遗漏（<5期）: 12个号码
- 温遗漏（5-15期）: 15个号码
- 冷遗漏（>15期）: 6个号码

⚠️ 注: 遗漏值不影响未来开出概率
```

### 4️⃣ 固定号码分析

**功能**: 分析用户指定的固定号码组合

**输出内容**:
- 各号码的当前状态（遗漏值、冷热情况）
- 组合合理性分析（奇偶比、大小比、区间分布）
- 建议的补充号码组合

**示例对话**:
```
用户: 分析固定号码 07 18 25 + 14

Agent:
## 固定号码分析报告

### 您的固定号码
红球: 07 18 25
蓝球: 14

### 号码状态
- 07号: 当前遗漏3期，近100期出现15次（热号）✓
- 18号: 当前遗漏0期，上期刚开出
- 25号: 当前遗漏5期，近100期出现12次
- 14号: 当前遗漏2期，近100期出现8次

### 组合分析
- 奇偶比: 2:1（建议补充1奇2偶）
- 大小比: 2:1（建议补充1大2小）
- 区间分布: 一区间1个，二区间2个
- 跨度: 18（偏小）

### 建议组合（娱乐）
1. 07 18 25 12 30 33 + 14
   说明: 补充热号12、30和33号
   
2. 07 18 25 03 11 29 + 14
   说明: 补充冷号11和遗漏较大的29号

⚠️ 注意: 以上仅为娱乐性质的随机组合
```

## ⚠️ 注意事项

### 1. 数据时效

- 数据更新可能存在延迟（通常30分钟内）
- 请以官方开奖结果为准
- 中奖请以官方公告为准

### 2. 分析性质

- 所有分析仅供娱乐参考
- 不构成投注建议
- 无法预测未来开奖

### 3. 理性购彩

- 设定购彩预算，不要超额
- 不要将购彩当作投资
- 未成年人不得购彩
- 如有成瘾迹象请寻求帮助

### 4. 技术限制

- 需要网络连接获取最新数据
- 数据准确性依赖官方数据源
- 网站结构变化可能影响数据获取

## 🆘 常见问题

### Q: 为什么历史数据不能预测未来？

A: 彩票开奖是完全随机的独立事件，每一期的开奖结果都不受历史影响。就像抛硬币，连续10次正面后，第11次正面的概率仍然是50%。

### Q: 热号真的更容易中奖吗？

A: 不是。热号只是统计上过去出现较多，但每个号码在每期中奖的概率完全相同。热号不会"冷却"，冷号也不会"反弹"。

### Q: 机选和自选哪个更好？

A: 从概率上来说完全相同。机选和自选中奖概率一样，选择哪种方式完全是个人喜好。

### Q: 系统能保证中奖吗？

A: 绝对不能。没有任何系统或方法能够保证彩票中奖。如果有人声称可以预测中奖号码，那一定是骗局。
